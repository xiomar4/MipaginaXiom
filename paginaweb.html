En la página web que podrá ver en el enlace a continuación, está todo el desarrollo de nuestro proyecto llamado X MEMORY. Nuestra propuesta se centra en un juego innovador y accesible desarrollado con Arduino Nano, con una interfaz intuitiva que facilita el entretenimiento sano y el ejercicio mental para usuarios de todas las edades. También puedes encontrar el proceso de desarrollo, por ejemplo: la programación detallada de Arduino en Tinkercad y la lista de componentes electrónicos para asegurar un rendimiento óptimo, priorizando la simplicidad y la efectividad de nuestro juego X MEMORY
